## Usage
* Clone repository<br>
* Install Node.js<br>
* Create an `.env` file and copy the contents of `.env-sample`. <br>
```
  cp .env-sample .env
```
* Run following commands<br>
```
  npm install
  npm run dev
```

* Before deploying, run command for production.<br>
```
  npm run prod
```
